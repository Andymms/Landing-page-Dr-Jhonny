# 🩺 Dr. Jhonny Mujica - Landing Page

A modern, fast, and fully responsive single-page web application built for **Dr. Jhonny Mujica** (Caracas, Venezuela). This professional landing page is optimized for medical consultations, highlighting patient care, professional background, and direct channels for scheduling appointments via WhatsApp and Instagram.

---

## 🛠️ Tech Stack

* **Frontend Library:** React.js
* **Styling & Framework:** Bootstrap 5 and CSS
* **Build Tool / Template:** 4Geeks Academy React Starter
