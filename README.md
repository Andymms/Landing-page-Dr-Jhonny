# 🩺 Dr. Jhonny Mujica - Landing Page Médica

¡Bienvenido/a! Este repositorio contiene el código fuente de la página web profesional desarrollada para el **Dr. Jhonny Mujica**, Cirujano Mastólogo basado en Caracas, Venezuela. 

El sitio ha sido diseñado meticulosamente con un enfoque mobile-first y optimizado para facilitar que los pacientes agenden consultas de manera rápida a través de integraciones directas con WhatsApp e Instagram.

🚀 **Ver sitio en vivo:** [https://landing-page-dr-jhonny.pages.dev/](https://landing-page-dr-jhonny.pages.dev/)

---

## 🎨 Características del Proyecto

- **Navegación Fluida (Scrollspy Nativo):** Estructura de una sola página (*Single-Page Application*) con transiciones animadas fluidas y un sistema de tracking activo en el Navbar adaptado a contenedores personalizados de React.
- **Diseño Premium y Corporativo:** Paleta de colores personalizada que combina tonos médicos clásicos, azules marinos profundos de alta autoridad y acentos brillantes para llamadas a la acción (CTA).
- **Favicon de Alta Definición:** Integración limpia en el core mediante un SVG codificado en URI de vectores nativos (Font Awesome) para máxima nitidez en pantallas de cualquier resolución (móviles hasta 4K).
- **Canales de Conversión Directa:** Burbuja informativa táctica y botones dinámicos conectados a APIs de mensajería instantánea para un flujo de reservas optimizado.
- **Arquitectura Modular:** Componentes limpios y reutilizables organizados lógicamente para un mantenimiento ágil.

---

## 🛠️ Tecnologías Utilizadas

- **Biblioteca Principal:** [React.js](https://react.dev/) (Componentes funcionales, Hooks de estado y efectos)
- **Framework de Estilos:** [Bootstrap 5](https://getbootstrap.com/) & CSS3 (Layouts responsivos, Grid System y Flexbox helpers)
- **Herramienta de Construcción:** [Vite](https://vitejs.dev/) (Entorno de desarrollo ultra rápido y empaquetado optimizado)
- **Iconografía:** [Bootstrap Icons](https://icons.getbootstrap.com/) y vectores inline
- **Control de Versiones:** Git & GitHub Codespaces
- **Plataforma de Despliegue:** Cloudflare Pages

## 📂 Estructura del Proyecto

```text
├── src/
│   ├── components/      # Componentes UI reutilizables (Navbar, Footer, etc.)
│   ├── views/           # Vistas y secciones principales (Hero/Inicio, Sobre Mí, Servicios, Contacto)
│   ├── styles/          # Hojas de estilo globales y variables CSS (index.css)
│   ├── main.jsx         # Punto de entrada de la aplicación React
│   └── index.html       # Plantilla base (contiene la optimización del favicon SVG)
├── package.json         # Dependencias y scripts de ejecución
└── vite.config.js       # Configuración del empaquetador Vite

```
